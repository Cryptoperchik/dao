# Chrome/Firefox 插件

[Liker Land Chrome/Firefox 插件](https://chrome.google.com/webstore/detail/liker-land/cjjcemdmkddjbofomfgjedpiifpgkjhe)讓你為網頁加上書籤儲存，留待有空時細閱。

Liker Land 是新世代的內容瀏覽器：

* 用戶可以全面控制閱讀的內容
* 每一次按讚都是對創作者的實際回報
