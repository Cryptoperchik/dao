# 修改頭像及顯示名稱

登入 [Liker Land](https://liker.land/) 並訪問 [https://like.co/in/settings](https://like.co/in/settings)，你會見到以下畫面。請點擊紅圈位置修改你的頭像及顯示名稱。

![](../../.gitbook/assets/edit-avator-displayname.png)

修改完成後，你的資訊會在 Liker Land 的各種使用場境中顯示。
